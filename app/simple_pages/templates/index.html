Proxmox

Proxmox is an open source hypervisor. Proxmox allows for the easy creation and management of virtual machines. Proxmox allows for creating both virtual machines and containers. Proxmox gives the ability to create VMs with any hardware configuration and emulation type. It also has more advanced features such PCI and PCIe passthrough which allows a virtual machine to take full control of the device passed. Proxmox has a built-in VNC viewer in order to view the VM as if it were its actual monitor. Other features include backup scheduling, HA clustering, and Ceph storage clustering.

MAAS

MAAS or Metal as a Service, is software made by Canonical (Developers of Ubuntu). MAAS allows for the adoption of hardware by inputting systems using their MAC address and selecting power type. This software is ment to be used with datacenter server hardware which has IPMI. IPMI allows for control of the server even if the server is turned off. This can be used to remotely turn on the server and monitor its statics. Once a machine is added into MAAS it will automatically turn the computer on and perform its tests using PxE booting. PxE booting is where the operating system is handed out over the LAN and installed. Once the computer is done running its tests the computer will then sit ready to be deployed. Once deployed the server will once again PxE boot and this time install the operating system chosen by the admin. In order for MAAS to work it needs to be the DHCP server for the network. It can also act as a local DNS server allowing each machine to have its on FQDN.

Portainer

Portainer is a docker container that allows for the easy administration of docker machines or clusters. Portainer can allow for multiple different administrators to manage the environments with different levels of permission. Portainer can allow for the advanced customization of docker environments with the ability to deploy prepackaged apps that only need a few items entered for configuration. Portainer allows for the easy access of logs of the containers and the ability to download them

Unifi

Unifi is a software network controller that allows for the administration of its hardware. Unifi is a relatively new company compared to the likes of Cisco, Juniper, MikroTIk, and TP-Link. Unifi allows users to host its network controller for free. The network controller is required to be able to manage Unifi's devices. Unifi does its network managing differently as it is mostly software defined networking. Setting up networks and vlans is as simple as following the setup wizards.

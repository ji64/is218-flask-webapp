Testing with Pytest

Testing with Pytest is very useful to make sure that components did not break and that everything is working as intended. In order to run the test open up a terminal inside the container and execute pytest -vv this will run all tests created in the tests folder inside the project.

To test that each page is accessible this test is executed to make sure its accessible and has the Section title on there. Here is an example of that such test for the Git page.

def test_request_page1(client):
    """This checks page 1"""
    response = client.get("/page1")
    assert response.status_code == 200
    assert b"Git" in response.data

Simple Pages

Using Simple Pages allows creating a base HTML file which have the navbar and footer then the different content pages are put in the middle allowing for a uniform page look without having to copy and paste code everywhere. It makes editing the navbar easy instead of having to change every file, only one file needs to be changed which is the file that has the HTML for it.
